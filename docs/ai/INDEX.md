# CineVerse AI Intelligence Hub

Welcome to the comprehensive technical documentation hub for **CineVerse's Artificial Intelligence Architecture**.

---

## 📚 Topics & Deep Dives

1. **Groq LLM Acceleration**: Sub-second LLaMA 3.3 70B inference for recommendations and JSON structured parsing.
2. **LangGraph Agent Orchestration**: State graph routing multi-turn user intent across specialized agents.
3. **LangChain & RAG Pipeline**: Retrieval-augmented generation for movie metadata and film trivia.
4. **FAISS & Embedding Vectors**: Vector similarity searching for film genome lookup.
5. **Prompt Engineering Design System**: Strict system prompts enforcing structured Zod outputs.
6. **Movie DNA Visual Genome**: Quantitative taste radar calculation (Mood, Tempo, Complexity, Visuals, Depth).
7. **Semantic Query Expander**: Translating natural language queries into TMDB filter criteria.
